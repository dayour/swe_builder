Skills

Skills are slash-invokable workflows. Eleven drive the agent build lifecycle. Two are utilities for filing bugs and feature suggestions to Azure DevOps.

Workflow skills (11)

/mcs-init Init

Create the project folder structure and a starter brief.json. Detects existing SDR documents in the folder and converts .docx to .md via pandoc.

/mcs-init MyAgent

Reads: project name. Writes: Build-Guides/MyAgent/{docs,agents}/ and starter brief. Swarms: none.

/mcs-context Context

Pull all M365 history about a customer via WorkIQ MCP. Searches emails, meetings, documents, Teams chats, and people. Pre-fills 60 to 80 percent of research.

/mcs-context Contoso

Reads: customer name. Writes: customer-context.md and customer-interactions.csv. Prerequisite: WorkIQ CLI authenticated.

/mcs-research Research

Read project documents, identify agents, research MCS components, design architecture, enrich brief with eval sets. Smart at both project and agent levels (Phase 0 detects new docs, brief edits, and manually created agents).

/mcs-research MyAgent
/mcs-research MyAgent agent-name

Reads: docs/, customer-context.md, knowledge/cache/, knowledge/learnings/. Writes: enriched brief.json, evals.csv. Swarms: Research Analyst (on demand), Prompt Engineer, Topic Engineer, QA Challenger.

/mcs-build Build

Construct and publish the agent in Copilot Studio using the hybrid stack. Includes account/environment gate, find-or-create logic, step-level checkpoints for resume, MVP filtering, eval-driven loop (critical gate, per-capability iteration, regression suite), and a QA validation gate before report.

/mcs-build MyAgent agent-name

Reads: brief.json. Writes: brief.json.buildStatus, MCS agent. Swarms: Topic Engineer, QA Challenger (plus Research Analyst and Prompt Engineer on demand).

/mcs-eval Quality

Run eval sets via Direct Line API (preferred), Playwright Test Chat (fallback), or Native MCS Eval (async). Per-set pass logic. Writes per-test results to brief for dashboard display.

/mcs-eval MyAgent agent-name
/mcs-eval MyAgent agent-name --set critical,functional
/mcs-eval MyAgent agent-name --native

Reads: brief.json evalSets. Writes: brief.json.evalSets[].tests[].lastResult, evals-results.json. Swarms: QA Challenger.

/mcs-fix Repair

Post-deployment fix loop. QA Challenger classifies failures into root causes, then PE and TE patch in parallel and re-evaluate. For initial build iteration, use /mcs-build (which has its own internal fix loop).

/mcs-fix MyAgent agent-name

Reads: brief.json (failing eval tests). Writes: brief.json (instructions, topics, fix history), MCS agent updates. Swarms: QA Challenger, Prompt Engineer, Topic Engineer.

/mcs-deploy Deploy

Promote agent dev to prod. Two modes: agent-level (fast, replicate-agent.js) and solution-level (PAC CLI export and import, ALM-ready). Includes pre-deploy validation, connection mapping, and post-deploy smoke test.

/mcs-deploy MyAgent agent-name

Reads: brief.json. Writes: deployment-report.md. Swarms: none.

/mcs-report Report

Generate customer-shareable reports from brief.json. Four report types for different audiences. Optional GPT review via the multi-model reviewer for second-opinion quality scoring.

/mcs-report MyAgent agent-name --type brief
/mcs-report MyAgent agent-name --type build
/mcs-report MyAgent agent-name --type customer
/mcs-report MyAgent agent-name --type deployment

Reads: brief.json. Writes: {type}-report.md.

/mcs-retro Retro

Post-session retrospective. Captures learnings into the knowledge index with Tier 1 (auto) and Tier 2 (user-confirmed) classification. Syncs upstream changes from tracked repos.

/mcs-retro MyAgent agent-name

Reads: session events, knowledge/learnings/index.json. Writes: learnings topic files, updated index.

/mcs-refresh Knowledge

Refresh one or all knowledge cache files with live research from MS Learn, web search, and MCS UI snapshots. Use to keep inventories current.

/mcs-refresh                      # all
/mcs-refresh triggers              # one file
/mcs-refresh --tier 1              # tier-1 (build-critical) only

Reads: knowledge/cache/. Writes: refreshed cache files with new last_verified.

/mcs-library Library

Browse, search, analyze, and contribute to the team's SharePoint solution library. Uses SWE-Swarms for intelligent tagging and pattern extraction.

/mcs-library list
/mcs-library search "knowledge agent"
/mcs-library upload MyAgent agent-name

Utility skills (2)

/bug ADO

Conversational bug filing. Gathers details, previews, then creates an ADO work item in powercatteam/FDE via az boards work-item create. Auto-enriches with project, agent, page, build status, and eval score from the dashboard context.

/suggest ADO

Conversational feature suggestion. Same dialog flow as /bug. Creates an ADO Feature work item.
